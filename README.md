# Ecommerce
#### Proyecto de una e-commerce de CoderHouse

Esta ecommerce todavia puede seguir en desarrollo

# Para correr localmente
```sh
git clone https://github.com/frncabre/EntregaFinal-Cabrera.git
cd ecommerce-cabrera
npm install
npm run dev
```
